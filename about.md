# Lagspike!
This mod displays a real-time graph showing when your game stutters or experiences frame timing issues.

## Features
- See visual frame timing graph (green = stable, red = stutter)
- Jitter measurement (how inconsistent your physics is)
- Stability percentage (what % of frames are on-time)
- Status indicator (GREEN if smooth, RED if laggy)

## Why Use It?
- **Prove it's not you** - Show your physics is stable in recordings/streams
- **Detect stutters** - See exactly when lag spikes occur
- **Monitor stability** - Know if your system is capable of smooth gameplay

The overlay shows:
- **[STABLE]** or **[JITTER!]** in green/red
- Current jitter value (lower is better)
- % of frames that are stable
- A graph of the last 60 physics frames

**That's it!** No configuration needed.
